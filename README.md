# Low Level Design

## Design Patterns

Implementing all the design patterns from the book [Head First Design Patterns](https://wickedlysmart.com/head-first-design-patterns) using C++.

### Running the code
```bash
cd design_patterns/${pattern_name}
fd -e cpp -e hpp -X g++
./a.out
```

### Cleanup
```bash
fd -e out -e gch -I -x rm
```
